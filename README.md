
# FeurBot

Un bot qui répondra Feur à tout message contenant "quoi" et de futur ajouts seront prévus.


## Features

- Auto reply "feur"


## Next features

- Auto create "channel-logs" on guildCreate
- Leaderboard


## Installation

Install modules

```bash
  npm install
```
    
## Start 

Start bot

```bash
  node .
```
